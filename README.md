# android-cli-programs
Applications for the command line on old ARM Android - Busybox, BASIC, LISP and TELNETD - in order to have a usable system even in most primitive old devices

XLISP came from:

https://almy.us/xlisp.html

The BAS BASIC interpreter came from previous versions of:

https://www.moria.de/~michael/bas/

Busybox being a variant of:

https://busybox.net/
